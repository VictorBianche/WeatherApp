# WeatherApp


Projeto Site Climatico

Este projeto está sendo desenvolvido visando melhoria e aprofundamento de conhecimentos.
O Inuito final é de que seja um site responsivo, que consiga trazer as informações climaticas da posição atual do usuario, trazendo junto consigo uma interação visual do clima atual.

As tecnologias e conceitos que estão sendo/serão utilizadas são:
-Javascript 
-HTML 
-CSS
-Git
-APIS
